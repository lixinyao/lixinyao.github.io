向上沟通

发表于 2019-12-23 | 分类于 沟通

职场上有两种人，一种是在等待机遇，一种是在创造机遇。创造机遇的人更主动，无疑要比等待机遇的人发展更好。经常在向上沟通中表现好的人，创造机遇的能力会更强。以下会教大家如何向上沟通

哪些因素影响了向上沟通的效果

有效的向上沟通 = 基本实力 * 互信状态

没啥实力还整天向上沟通，老板会觉得你在瞎BB；有实力，但不和老板沟通，老板会觉得你很高冷，也很难得到老板的信任。只有基本实力和互信状态都具备，才能有效的向上沟通。

基本实力是大家都了解的一些硬实力，比如智力、体力、敬业等等。互信状态则是一些软实力，那我们怎么样才能建立互信状态呢？

  1. 思维同频。向老板一样思考，帮助老板化解难题
  2. 沟通同频。了解老板是什么类型的（随和？细节？），平时注意沟通技巧
  3. 行动同频。要用贡献建立信任，帮助老板打胜仗

向上沟通的常见场景

  1. 接受任务。利用5W+2H使任务明确
    • why。背景是啥，目标是啥
    • what。做什么事情
    • when。什么时候完成
    • who。合作方是谁
    • where。地点及场地要求
    • how。怎么样完成
    • how much。数量及质量要求
  2. 请示任务。用3S法则和老板对齐，保证方向正确
    • situation。汇报目标及主要问题，做了这件事会有什么价值
    • solution。解决方案是什么，需要什么资源及支持
    • suggestion。请求老板批复
  3. 执行任务。
    • 阶段性汇报。逻辑清晰、积极主动
    • 寻求帮助。不把难题抛给老板，要带着解决方案
    • 被批评。保持理性、不解释、事后总结
  4. 完成任务。强调功劳而非苦劳，强调价值而非事务。可以利用以下工具更高效汇报
    • 黄金圈。为什么做->怎么做->做了什么
    • 空雨伞。现状->预测->方案（天上有云，预测有雨，我要带伞）
    • 时间轴。随着时间推进，有了哪些改善
    • 金字塔。中心论点->论据（独立且不遗漏）
