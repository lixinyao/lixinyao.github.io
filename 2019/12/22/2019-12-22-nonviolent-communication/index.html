《非暴力沟通》

发表于 2019-12-22 | 分类于 沟通

在日常生活和职场中，很容易由于沟通技巧的问题，双方搞的不欢而散。《非暴力沟通》介绍了一种沟通方式，即如何转变谈话和聆听的方式来减少人与人之间因为语言带来的痛苦。作者相信，人天生热爱生命、乐于互助，但是语言及表达方式却使得日常生活中充满了痛苦，他提出了“非暴力沟通”这种沟通方式，来明确自己和他人的观察、感受、需求和请求，进而达成双方之间的情意相通、乐于互助

让爱融入生活

非暴力沟通的4个要素：

  1. 观察
  2. 感受
  3. 需要
  4. 请求

这种沟通方式的效果显而易见，可以用两组对话感受暴力沟通和非暴力沟通的差异：

  • 暴力沟通
    A:“你怎么这么懒呢，东西乱七八糟的也不收拾？”
    B:“…”
  • 非暴力沟通
    A:“我看到屋子比较乱，我有点生气，我希望屋子是整洁的，你能收拾一下吗？”
    B:“好的”

用非暴力沟通的4个要素组织的沟通，确实能提高沟通的质量

是什么蒙蔽了爱？

异化的沟通方式使我们难以体会心中的爱。以下几种沟通方式都是异化的沟通方式：

  1. 道德评判。例如“你太自私了”很可能招致敌意
  2. 进行比较。例如“你没有小王聪明”，不会让对方有好的感受
  3. 回避责任。例如领导问“为什么迟到了”，你说“今天没赶上公共汽车，导致晚了”。这样领导感受就不好，他想的是难道你不能早起几分钟吗？
  4. 强人所难。例如亲人突发重病，你想请假回去探望。领导说“今天工作比较重要，我希望你在工作岗位上”

大多数人的语言倾向于批判、比较、命令和指责，而没有倾听彼此的感受和需要。换位思考一下，就能体会到对方的感受

区分观察和评论

非暴力沟通的第一个要素是观察。将观察和评论混为一谈，别人就会倾向于听到批评，并反驳我们

观察是完全客观的描述，而评论会带有个人情感。举个例子感受下区别：“我爸爸是个好人”是评论；而“我爸爸昨天给灾区捐款1000”是观察

体会和表达感受

非暴力沟通的第二个要素是感受。通过建立表达感受的词汇表，我们可以更清楚的表达感受，从而使沟通更为顺畅

几个例子区分想法和感受：

  • 我觉得你不爱我。想法
  • 你要离开，我很难过。感受
  • 当你说那句话时，我感到害怕。感受
  • 如果你不和我打招呼，我会觉得你不在乎我。想法
  • 你能来，我很高兴。感受
  • 你真可恶。想法
  • 我想打你。想法
  • 我觉得我被人误解了。想法
  • 你帮我的忙，我很开心。（书上说是想法，但我理解是感受）
  • 我是个没用的人。想法

感受的根源

感受的根源来源于自身的需要而非他人的言行，我们可以体会两种心理的差异：

  • 你昨晚没来令我很失望（内心认为不好的感受是他人带来的）
  • 昨晚你没来，我很失望，因为我想和你说一些烦心事（感到失望是由于自己的愿望没有满足）

如果我们通过批评来提出主张，人们的反应常常是申辩或反击；如果直接说出我们的需要，则较有可能得到对方的积极回应

几个例子看一下发言者是否对自己的感受负责（说出自己的具体需要才是负责）

  • 你将公司机密文件放在了会议室，太令我失望了。（不负责）
  • 你这么说，我很紧张。我需要尊重。（负责）
  • 你来的这么晚，让我很郁闷。（不负责）
  • 你无法来吃晚饭，我很难过。我本来想和你好好聊一聊。（负责）
  • 我很伤心。因为你没有做你答应我的事情。（不负责）
  • 我很沮丧。我希望我的工作已经取得更大的进展。（我理解不负责，感觉需要不具体，但书上为负责）
  • 朋友叫我外号让我很难过。（不负责）
  • 你得奖了，我很高兴。（不负责）
  • 你的嗓门那么大，吓死人了（不负责）
  • 你让我搭你的车回家，我很感激。因为我想比孩子们先到家（负责）

请求帮助

我们提出的请求越具体越好。如果我们的意思含糊不清，别人就难以了解我们到底想要什么。表达了请求后，我们还需要请求对方的反馈。
命令会给对方不好的感受。批评指责及利用对方的内疚达到目的，都是命令的方式而不是请求

几个例子看一下哪些提出了明确的请求：

  • 我希望你理解我（否）
  • 请告诉我，在我做的事情中，你最满意的是哪一件（是）
  • 我希望你更加自信（否）
  • 不要再喝酒了（否）
  • 请让我成为我自己（否）
  • 关于昨天的会议，请不要隐瞒你的看法（否，不要隐瞒太模糊，换成给我一些建议会更好）
  • 我希望你能在规定时速驾驶（是）
  • 我想更好的了解你（否）
  • 我希望你尊重我的个人隐私（否）
  • 我希望你经常做晚饭（否）

用全身心倾听

前面讲的是对于我们自己，如何表达观察、感受、需要、请求。本节主要是如何倾听他人的观察、感受、需要和请求
当他人遭遇不幸时，我们不要急于表达建议、态度、感受，我们需要用心的倾听对方，这样有助于他人的理解和接纳。在有的时候，我们可以主动表达自己的理解，来帮助对方了解我们在何种程度明白了他的意思

不用心体会感受和需要的例子：
A：我又误机了，我真是个混蛋！
B：没有人是十全十美的，不要太严格要求自己
（B没有了解A的需要，着急的提出安慰和建议）

用心体会感受和需要的例子：
A：你以为你什么都知道？
B：听起来，你有些不耐烦，因为你希望每个人的意见都能得到倾听？

倾听的力量

倾听能够使我们勇于面对自己的弱点，帮助我们预防潜在的暴力，治愈心灵的创伤

爱自己

非暴力沟通最重要的应用也许使培育对于自己的爱。评价自己的行为时，我们专注于尚未满足的需要，这样我们就不再依赖羞愧、内疚、恼怒、沮丧等心理来寻求改变，而让爱主导我们的学习和成长
通过深入理解我们行为的动机，并用“选择做”来取代“不得不”，我们的生活将变的和谐并快乐

  1. 列好一个清单，哪些事情觉得没意思，又不得不做的
  2. 做这些事情，是我选择做，而不是不得不做。在清单的每一句都加上“我选择做”
  3. 填写声明，比如“我选择做XXX是因为我想要XXX”

充分表达愤怒

在生气时，批评和指责他人无法真正的传达我们的心声，直接说出我们的需要更有可能使我们的愿望得到满足
表达愤怒的4个步骤：

  1. 停下来，除了呼吸，什么都别做
  2. 想一想是什么想法让我们生气了
  3. 体会我们的需要是什么
  4. 表达感受和尚未满足的需要

运用强制力避免伤害

运用强制力是为了保护自己和他人，而不是惩罚他人。如果我们威胁他人或实施惩罚，人们常常会产生敌意和抵触心理，这样彼此的关系就会疏远。运用惩罚来使人们认识自己的需要，很可能适得其反

重获生活的热情

通过运用非暴力沟通，我们不再试图分析自己或他人有什么毛病，而是用心去了解我们的需要，这样我们的内心将变得平和。一旦我们发现自己心底深处的愿望，并采取积极的行动，我们将会重获生活的热情

表达感激

非暴力沟通鼓励我们充分的表达感激。在表达感激时，我们说出：

  1. 对我们有益的行为
  2. 我们的哪些需要得到了满足
  3. 我们的需要得到满足后，我们是什么样的心情
